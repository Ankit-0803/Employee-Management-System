# **Employee Management System**  

This project is a web-based application developed using **React** with **SweetAlert2** for interactive alerts and **Jest** for robust testing.  
The system ensures a smooth and intuitive user experience while optimizing workflows for managing employee data effectively.  

---

## **Project Overview**  

The **Employee Management System** streamlines employee data management with a user-friendly interface and advanced features.  
By leveraging modern technologies, the project:  
- Achieves **90% compatibility** across modern browsers, including **Chrome**, **Firefox**, and **Safari**.  
- **Reduces load times** by 30% with efficient build and deployment processes.  
- Enhances **data entry accuracy** by 25% through real-time validation and error handling.
